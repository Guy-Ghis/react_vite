# React + Vite

---

## My App

A fast, modern React project bootstrapped with [Vite](https://vitejs.dev/) for lightning-fast development and build performance.

## 🚀 Project Setup

This project was created using:

```bash
npm create vite@latest my-app -- --template react
```

## 📦 Tech Stack

- ⚛️ React 18+
- ⚡ Vite
- 🛠️ JavaScript (or TypeScript, if applicable)
- 🎨 Optional: Tailwind CSS, React Router, etc.

## 🛠️ Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/Guy-Ghis/react_vite.git
cd react_vite
npm install
```

## 🧪 Run the Development Server

```bash
npm run dev
```

Vite will start a development server at:
[http://localhost:5173](http://localhost:5173)

## 🔨 Build for Production

```bash
npm run build
```

This will generate a production-ready build in the `dist/` folder.

## 🧹 Preview the Production Build

```bash
npm run preview
```

This simulates serving the `dist/` folder with a local static server.
